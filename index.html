HTML Introductión

¿Que es HTML?

  • Standar for Hyper Text Markup Language
  • Desribes the structure of a Web Pages
  • Consists of a series of elements
  • Elements tell the browser how to display the content
  • Elements consists on pieces label with content

Basics Elements Explain

  1. La etiqueta "DOCTYPE" define la declaracion del documento en cuestion como HTML
  2. La etiqueta "head" contiene informacion adicional acerca de la pagina HTML
  3. La etiqueta "HTML" es el elemento principal de una pagina HTMl
  4. La etiqueta "title" especifica el titulo de la pagina web
  5. La etiqueta "body" especifica el cuerpo de la pagina web (todos los elementos visibles)
  6. La etiqueta "h1" define un texto de tamaño titular
  7. La etiqueta "h2" define un texto de tamaño subtitular
  8. La etiqueta "p" define un texto de tamaño

¿Que es un elemento HTML?

Un elemento HTML esta definido por una etiqueta de inicio, seguido de algún contenido y una etiqueta de finalización
  • <nombreEtiqueta>Etiqueta de inicio
  • Contenido: entre la etiueta de inicio y la eitqueta de finalizacion
  • <nombreEtiqueta>: etiqueta de finalizacion

Etiqueta <map>

La etiqueta <map> define el mapa de una imagen. Una imagen mapeada es una imagen que tiene distinatas areas detectables por el naveagador, logrando identificar de manera independiente cada una de las areas. La etiqueta <map> define las areas de la imagen mediante la etiqueta <area> con los atributos HTML "shape" que define la forma del area señalada en la imagen,"coords" define las coordenadas de cada uno de los puntos de la shape, "href" puede definir un hipervinculo para cada uno de las areas.
Ejemplo:
<img src="imagen1.jpg" usemap="#workmap">
<map name="imagen1">
<aerea shape = "rect" coords = "34,44,270,350" href = "pagina1.com">
<aerea shape = "circle" coords = "200,172,333,250" href = "pagina2.com">
</map>
